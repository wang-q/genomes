# Genomes

## Software

```shell
cbp install nwr
cbp install sqlite3 # 3.34 or above

cbp install hmmer easel
cbp install mafft muscle trimal
cbp install samtools
cbp install fasttree iqtree2
cbp install mash mmseqs

cbp install fd pigz
cbp install jq tectonic
cbp install tva 
cbp install pgr
```
